Installation Instructions

  1. In Visual Studio Code, Click the extensions icon in the left column toolbar.
  2. In the search bar that appears, type Live Server.
  3. Select the live server extension. it should be top choice
  4. In the page the Live Server appears click the green install button.
id="use">

Usage Instructions

  • Visual studio code must be open to a folder to use Live Server.
  • In VS code explorer, right click (Windows) or control click (Mac) on the HTML file you want to open. From the context menu, select "Open with Live Server".
